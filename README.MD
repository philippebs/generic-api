# Flask example

Usando Flask para verificar se o servidor está online

Integração com Flask-Cors.


## Installation

Install with pip:
```
$ pip install -r requirements.txt
```

## Flask Application Structure 
```
.
| |────resource/
| |────templates/
| | |────__init.py
| | |────index.html
| | |────servers.html
|──────model.py
|──────app.py
|──────util.py

```
